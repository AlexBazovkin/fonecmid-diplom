**Настройка телеграм бота и группы:**

1. Откройте Telegram (https://t.me/BotFather) и найдите @BotFather.
Это бот для создания новых и управления созданными ранее ботами.
2. Отправьте команду /start, чтобы начать взаимодействие с
@BotFather. Он предоставит список доступных команд.
3. Для создания нового бота отправьте команду /newbot. @BotFather попросит ввести имя и уникальное имя пользователя для бота. Имя
пользователя должно заканчиваться на «bot». Если имя пользователя
уникальное, вы получите сообщение, что бот успешно создан, и увидите
токен.
4. Сохраните токен управления телеграмм ботом, который пришлет
BotFather, поскольку он понадобится в дальнейшем для заполнения значения
константы доработки.

**Создание группы в мессенджере Телеграм:**


1. Создайте произвольную группу в Телеграм.
2. Добавьте в группу ранее созданного бота.
3. Назначьте боту права администратора.
4. Напишите любое сообщение в группу.
5.  В строку браузера вставьте адрес 
«https://api.telegram.org/bot[ВашТокен]/getUpdates» и выполните запрос. В полученных данных
найдите идентификатор группы для оповещения (начинается с "-", пример: -1234567891234)
и сохраните его,
поскольку он понадобится в дальнейшем для заполнения значения константы
доработки.
